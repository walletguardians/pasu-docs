# \[NFT] NFT 구매 및 전송 관리

NFT 구매를 할 때에는 대금을 치렀더니 NFT 토큰이 남에게 가버라거나, 바가지를 쓰는 사고가 흔히 발생합니다. 구매 NFT의 수령처와 지불 금액, 전송 수령처를 검사하는 패키지입니다.

### 이 패키지로 막을 수 있는 것

* NFT를 소각 주소나 블랙리스팅된 주소로 전송하는 상황
* NFT를 구매하는 경우, NFT가 본인이 아닌 주소로 가는 상황
* 터무니없이 비싸게 NFT를 사는 상황
