# \[NFT] NFT 판매 관리

NFT 리스팅 사기는 정상처럼 보이는 주문에 대금을 받는 주소를 바꾸거나 헐값에 판매하고, 컬렉션에 대한 권한 위임을 숨겨 둡니다. 판매 대금의 수령처와 금액, 주문 유형과 유효기간을 검사해, 서명 한 번에 NFT를 잃는 사고를 막는 패키지입니다.

### 이 패키지로 막을 수 있는 것

* 판매 대금이 본인에게 오지 않는 주문에 서명하는 상황
* 최저가보다 크게 낮은 값에 NFT를 파는 주문에 서명하는 상황
* 컬렉션을 통째로 넘기는 주문에 서명하는 상황.
* 서명할 일이 없는 비정상 주문 유형에 서명하는 상황
* 유효기간이 비정상적으로 긴 주문에 서명하는 상황
