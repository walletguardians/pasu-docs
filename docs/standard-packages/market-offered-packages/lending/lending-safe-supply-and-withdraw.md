# \[대출] 예치 & 출금 기본 보호

예치/출금을 사용하는 사용자가 처할 수 있는 위험은 엉뚱한 풀, 엉뚱한 수취인, 엉뚱한 금액과 상호작용하는 데에서 옵니다. 이를 예치와 출금하는 상황에 검사하는 패키지입니다.

### 이 패키지로 막을 수 있는 것

* 예치가 다른 계정 명의(onBehalfOf)로 잡히는 상황
* 다른 주소의 빛을 상환하는 상황
* 동결되거나 정지된 풀에 예치해 돈이 묶이는 상황
* 한 번에 지나치게 큰 금액을 예치하는 상황
* 출금한 돈이 내 지갑이 아닌 주소로 가는 상황
* 빚이 있는 상태에서 담보 전액 출금하는 상황

### 정책 목록

* 예치가 다른 계정 명의로 등록될 시 경고 (LEND-003)
* 내가 아닌 다른 주소의 빚을 상환할 시 경고 (LEND-005)
* 동결되거나 정지된 자산 풀에 예치 요청 시 경고 (LEND-006)
* 일정 금액(USD)을 초과하는 예치 요청 시 경고 (LEND-007)
* 출금 자산이 제3자에게 갈 시 경고 (LEND-008)
* 부채가 있는 상태에서 담보 전액 출금 요청 시 경고 (LEND-009)
