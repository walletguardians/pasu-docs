# \[Lending] Liquidation Guard

### 이 패키지로 막을 수 있는 것

담보 대출을 쓰는 사용자가 차입, 출금, 담보 설정을 바꿀 때, 포지션이 청산 위험에 빠지지 않도록 미리 이상 징후를 파악해 경고합니다.

* 차입 직후 Health Factor(HF)가 위험 수준(1.5 미만)으로 떨어지지 않도록 경고합니다.
* 담보를 출금함으로써 포지션이 위험해지거나 청산되지 않도록 경고합니다.
* 빚이 있는 상태에서 담보를 비활성화(disable collateral) 않도록 경고합니다.
* 빚이 있는 상태에서 청산의 기준선을 바꾸는 이모드(e-mode(Aave)) 변경을 모르고 하지 않도록 경고합니다.
* 동결되거나 정지된 자산 풀에서 차입하지 않도록 경고합니다.

### 상세 설명

담보를 넣고 빌리는 사용자가 스스로 포지션을 청산 위기로 모는 위험한 차입, 위험한 출금, 담보 해제, 이모드 변경, 동결된 풀에서의 차입이라는 5가지 동작을 서명 직전에 잡아 구성한 패키지입니다.

### 정책 목록

* 차입 후 청산 여유(HF)가 기준 미만으로 떨어질 시 경고
* 담보 출금 후 청산 여유(HF)가 기준 미만으로 떨어질 시 경고
* 부채가 있는 상태에서 담보 사용 해제 요청 시 경고
* 부채가 있는 상태에서 이모드(e-mode) 변경 요청 시 경고
* 동결되거나 정지된 자산 풀에서 차입 요청 시 경고
