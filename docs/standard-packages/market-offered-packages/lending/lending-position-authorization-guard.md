# \[대출] 권한 위임 기본 보호

한 번의 서명으로 포지션 전체의 통제권을 넘길 수 있는 권한 위임 동작은 한번 잘못 서명시 큰 위험이 발생합니다. 권한 위임을 관리할 수 있도록 하는 패키지입니다.&#x20;

### 이 패키지로 막을 수 있는 것

* 내 모든 포지션을 통제할 권한(setAuthorization)을 넘기는 상황
* 악성으로 알려진 주소에 포지션 권한을 부여하는 상황
* 예치를 할 때 예치한 자산이 다른 계정 명의로 잡히도록 권한을 부여하는 상황
* 차입이 다른 계정 명의로 잡히도록 하는 권한을 부여하는 상황
* 다른 계정의 부채를 대신 상환하도록 권한을 부여하는 상황
