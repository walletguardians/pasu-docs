# \[Lending] Position Authorization Guard

### 이 패키지로 막을 수 있는 것

렌딩 프로토콜의 권한 위임과 신용 위임을 쓰는 사용자가, 포지션에 대한 권한을 엉뚱한 주소에 넘기지 않도록 미리 이상 징후를 파악해 경고합니다.

* 내 모든 포지션을 통제할 권한(setAuthorization)을 넘기기 전에 한 번 더 확인하도록 경고합니다.
* 악성으로 알려진 주소에 포지션 권한을 부여하지 않도록 차단합니다.
* 차입이 다른 계정 명의로 잡히지 않도록 경고합니다.
* 남의 빚을 모르고 대신 갚지 않도록 경고합니다.
* 예치를 할 때 예치한 자산이 다른 계정 명의로 잡히지 않도록 경고합니다.

### 상세 설명

한 번의 서명으로 포지션 전체의 통제권을 넘길 수 있는 권한 위임 동작만 따로 관리할 수 있도록 하는 패키지입니다.&#x20;

### 정책 목록

* 포지션 전체 통제 권한의 위임 요청 시 경고
* 악성으로 알려진 주소에 포지션 통제 권한 위임 요청 시 차단
* 차입이 다른 계정 명의로 등록될 시 경고
* 다른 계정의 부채를 대신 상환할 시 경고
* 예치가 다른 계정 명의로 등록될 시 경고
