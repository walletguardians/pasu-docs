# \[DEX] Swap 기본 보호

Swap을 처음 할 때 발생할 수 있는 주요한 사고들을 서명 전에 방지하고자 구성한 패키지입니다.

### 이 패키지로 막을 수 있는 것

손해 없이 토큰을 Swap할 수 있도록 이상 징후를 파악하면 경고를 표시합니다.

* Swap으로 받을 토큰을 제3자에게 보내는 행위
* 슬리피지를 과도하게 크게 허용한 Swap을 하는 행위
* 검증되지 않았거나, 악성 스캠 토큰과 Swap 하는 행위

### 정책 목록

* Swap으로 받을 자산이 제3자에게 갈 시 경고 (AMM-001)
* 슬리피지 허용치가 1%를 초과하는 Swap을 실행할 시 경고 (AMM-010)
* 스왑으로 받을 토큰이 스캠으로 분류될 시 경고 (AMM-012)
