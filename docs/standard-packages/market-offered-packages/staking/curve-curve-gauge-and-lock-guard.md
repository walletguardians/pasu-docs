# \[Curve] Curve Gauge & Lock Guard

### 이 패키지로 막을 수 있는 것

Curve의 veCRV 락과 게이지를 다루는 사용자의 예치된 자산과 투표권이 엉뚱한 곳으로 가지 않도록 미리 이상 징후를 파악해 경고합니다.

* 내 CRV가 다른 사람의 veCRV 락으로 들어가지 않도록 차단합니다.
* 게이지 예치가 본인이 아닌 주소로 적립되지 않도록 차단합니다.
* 게이지 가중치 투표를 할 때 대상과 가중치를 확인하도록 경고합니다.
* 다른 계정을 대리한 보상 청구를 모르고 하지 않도록 경고합니다.

### 상세 설명

Curve의 락과 게이지는 예치를 다른 사람 명의로 적립하거나 투표권을 잘못 행사하기 쉬운 구조입니다. 대신 예치, 게이지 적립, 게이지 투표, 대리 청구라는 신호를 검사하는 패키지입니다.

### 정책 목록

* 내 CRV가 다른 사람의 veCRV 락으로 들어갈 시 차단
* 게이지 예치가 본인이 아닌 주소로 적립될 시 차단
* veCRV 게이지 가중치 투표 시 대상과 가중치를 확인하도록 경고
* 다른 계정을 대리해 보상을 청구할 시 경고
