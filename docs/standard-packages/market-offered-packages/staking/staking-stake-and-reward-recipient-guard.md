# \[Staking] Stake & Reward Recipient Guard

### 이 패키지로 막을 수 있는 것

토큰을 스테이킹하고 보상을 받는 사용자가, 예치 지분과 보상이 남에게 가지 않도록 미리 이상 징후를 파악해 경고합니다.

* 스테이킹 지분이 본인이 아닌 주소로 발행되지 않도록 차단합니다.
* 스테이킹 보상이 본인이 아닌 주소로 가지 않도록 차단합니다.
* 언스테이킹으로 받을 자산이 본인이 아닌 주소로 가지 않도록 차단합니다.
* 슬래싱 위험이 있는 풀에 스테이킹할 때 위험을 미리 알리도록 경고합니다.

### 상세 설명

스테이킹은 예치 지분과 보상의 수령처가 분리되어, 그 수령처가 바뀌어도 모르고 서명하기 쉽습니다. 예치, 보상, 언스테이킹의 수령처가 본인인지 일관되게 검사하고, 슬래싱 위험 풀을 함께 알리는 패키지입니다.

### 정책 목록

* Aave 스테이킹 지분이 본인이 아닌 주소로 발행될 시 차단
* Ethena(sUSDe) 스테이킹 지분이 본인이 아닌 주소로 갈 시 차단
* Aave 보상 수령처가 본인이 아닐 시 차단
* Curve 게이지 보상 수령처가 본인이 아닐 시 차단
* Ethena 언스테이킹(USDe) 수령처가 본인이 아닐 시 차단
* 슬래싱 위험이 있는 풀에 스테이킹할 시 경고
