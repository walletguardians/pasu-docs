# \[Aave] Aave Safety Module Exit

### 이 패키지로 막을 수 있는 것

Aave 안전모듈처럼 인출에 쿨다운과 기간 제약이 있는 스테이킹을 쓰는 사용자가, 잘못된 타이밍에 인출하지 않도록 미리 이상 징후를 파악해 경고합니다.

* 언스테이킹 쿨다운을 시작할 때 좁은 인출 창을 미리 알리도록 경고합니다.
* 인출 창 밖에서 상환을 시도해 거래가 되돌려지지 않도록 경고합니다.
* 전액 상환을 모르고 하지 않도록 경고합니다.

### 상세 설명

쿨다운 기반 스테이킹은 인출 가능한 창이 좁아, 타이밍을 놓치면 다시 기다리거나 거래가 실패합니다. 쿨다운 시작, 인출 창, 전액 상환이라는 타이밍 신호를 검사하는 패키지입니다.

### 정책 목록

* 언스테이킹 쿨다운을 시작할 시 좁은 인출 창을 안내하며 경고
* 인출 창 밖에서 Aave 안전모듈 상환을 시도할 시 경고
* Aave 안전모듈에서 전액 상환할 시 경고
