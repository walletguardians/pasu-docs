# \[Bridge] Recipient Safety

### 이 패키지로 막을 수 있는 것

크로스체인 브릿지를 쓸 때, 보낸 돈이 정확히 내 지갑으로 도착하도록 미리 이상 징후를 파악해 경고합니다.

* 도착 체인의 소각 주소로 브릿지해 자산을 영영 잃지 않도록 차단합니다.
* 도착 주소가 본인 지갑이 아니면 한 번 더 확인하도록 경고합니다.
* EVM이 아닌 체인(예: Solana)의 raw 형식 주소로 브릿지할 때 확인하도록 경고합니다.

### 상세 설명

브릿지는 보내는 체인과 받는 체인이 달라 도착 주소를 한 번 잘못 적으면 되돌릴 수 없습니다. 도착 주소가 소각 주소인지, 본인 지갑이 맞는지, 검증하기 어려운 raw 형식의 비EVM 주소인지 짚어, 돈이 엉뚱한 곳으로 도착하는 사고를 막는 패키지입니다.

### 정책 목록

* 도착 주소가 소각 주소일 시 차단
* 도착 주소가 본인 지갑이 아닐 시 경고
* EVM이 아닌 raw 형식 주소로 브릿지할 시 경고
