# NFT-009: 마켓 주문 대금이 본인에게 들어오지 않을 시 경고

### Policy Definition (정책 정의)

> Seaport 주문을 서명할 때 그 대금이 **본인 주소로 한 푼도 들어오지 않으면** 경고를 표시합니다.

Seaport에서 NFT를 팔거나 오퍼를 낼 때는 주문에 서명을 하고, 거래가 체결되면 대금을 받을 주소가 그 주문 안에 적혀 있습니다. 정상적인 리스팅이라면 이 수령처는 판매자 본인입니다. 서명하려는 주문의 대금을 받을 주소가 본인이 아닐 경우 경고를 표시합니다.

#### Scope (적용 범위)

Seaport 마켓 주문에 서명할 때, 대금을 받을 주소에 본인이 빠져 있는 경우에 적용됩니다.

#### Audience (대상 사용자)

NFT를 리스팅하거나 오퍼를 내는 Seaport 사용자

#### Used Data (판정에 사용될 데이터)

* 체결하려는 주문의 마켓플레이스 프로토콜(seaport)
* 대금이 본인 주소로 들어오는지 여부

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("proceeds-not-to-self-warn")
@severity("warn")
@reason("이 주문의 대금이 본인 주소로 한 푼도 들어오지 않습니다 — 내놓은 자산을 주고 아무것도 받지 못하니 확인하세요")
forbid (
    principal,
    action == Marketplace::Action::"SignOrder",
    resource
)
when
{
    context.venue.name == "seaport" &&
    context.proceedsToOfferer == false
};
```
{% endcode %}
