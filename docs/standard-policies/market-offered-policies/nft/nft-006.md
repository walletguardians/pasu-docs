# NFT-006: Seaport 구매에서 바닥가의 50배가 넘는 금액을 지불할 시 경고

### Policy Definition (정책 정의)

> Seaport 주문을 체결해 NFT를 살 때 받는 컬렉션의 **바닥가(floor)** 대비 50배가 넘는 금액을 내면 경고를 표시합니다.

바닥가(floor)는 해당 컬렉션에서 지금 가장 싸게 나온 매물의 가격으로, 그 NFT의 대략적인 시세 하한입니다. Seaport에서 리스팅이나 오퍼를 체결해 NFT를 살 때, 내가 내는 금액이 받는 컬렉션의 바닥가보다 지나치게 높다면 가격을 잘못 봤거나, 시세를 부풀려 비싸게 떠넘기는 주문일 수 있습니다. 그래서 지불 금액이 컬렉션 바닥가의 50배를 넘으면 구매에 서명하기 전에 금액이 맞는지 확인하도록 경고합니다.

#### Scope (적용 범위)

Seaport 마켓플레이스에서 주문을 체결해 NFT를 구매하는 경우에 적용됩니다. (FulfillOrder, venue=seaport) 받는 컬렉션의 바닥가가 조회되고, 지불 금액이 그 50배를 넘을 때 발동합니다.

#### Audience (대상 사용자)

OpenSea 등 Seaport 기반 마켓플레이스에서 NFT를 구매하는 사용자

#### Used Data (판정에 사용될 데이터)

체결하려는 주문에서 받는 NFT(`offer`)와 내가 내는 대금(`consideration`), 그리고 받는 컬렉션의 바닥가 대비 지불 배수(`overpayMultiple` — 서버의 NFT 바닥가 오라클 조회 결과)

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("fulfill-overpay-vs-floor-warn")
@severity("warn")
@reason("받는 NFT의 바닥가(floor)보다 훨씬 비싸게 지불합니다 — 승인 전에 금액을 확인하세요")
forbid (
    principal,
    action == Marketplace::Action::"FulfillOrder",
    resource
)
when
{
    context.venue.name == "seaport" &&
    context has custom &&
    context.custom has overpayMultiple &&
    context.custom.overpayMultiple.greaterThan(decimal("50.0000"))
};
```
