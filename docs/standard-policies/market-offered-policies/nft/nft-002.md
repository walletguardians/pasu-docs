# NFT-002: 낯선 위임처에 NFT 컬렉션 전체 사용 권한을 줄 시 경고

### Policy Definition (정책 정의)

> 알려진 거래소 위임처가 아닌 곳에 컬렉션 전체를 옮길 권한을 주는 경우 경고를 표시합니다.

NFT의 `setApprovalForAll`은 특정 토큰 하나가 아니라 그 컬렉션에 든 NFT 전부를 상대가 마음대로 옮길 수 있게 위임하는 승인입니다. OpenSea나 Blur 같은 거래소에서 리스팅하려면 한 번씩 거쳐야 하는 정상적인 단계이지만, 같은 승인을 드레이너가 받아 가면 그 컬렉션의 NFT를 통째로 털 수 있습니다. 그래서 승인을 받아 가는 위임처가 널리 쓰이는 거래소의 것이 아니면, 정말 그 상대에게 컬렉션 전체를 맡길 것인지 한 번 더 확인하도록 알립니다.

#### Scope (적용 범위)

NFT의 `setApprovalForAll` 서명에 적용됩니다. 권한을 새로 주는 승인이면서, 받아 가는 위임처가 알려진 거래소(OpenSea Conduit, Blur ExecutionDelegate, LooksRare v2 TransferManager, X2Y2 ERC721Delegate)가 아닐 때 발동합니다.

#### Audience (대상 사용자)

NFT를 보유하거나 거래소에 리스팅하는 모든 Wallet 사용자

#### Used Data (판정에 사용될 데이터)

권한을 새로 주는 승인인지(`context.approved`), 그리고 승인을 받아 가는 위임처 주소(`context.spender`)

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("nft-setapprovalforall-conduit-warn")
@severity("warn")
@reason("내 컬렉션 전체를 옮길 권한을 주는데 알려진 거래소 위임처가 아닙니다. 승인 대상이 맞는지 확인해 주십시오.")
forbid(principal, action == Token::Action::"NftSetApprovalForAll", resource)
when {
  context.approved
  && !([
    "0x1e0049783f008a0085193e00003d00cd54003c71", // OpenSea Conduit
    "0x00000000000111abe46ff893f3b2fdf1f759a8a8", // Blur ExecutionDelegate
    "0x000000000060c4ca14cfc4325359062ace33fe3d", // LooksRare v2 TransferManager
    "0xf849de01b080adc3a814fabe1e2087475cf2e354"  // X2Y2 ERC721Delegate
  ].contains(context.spender))
};
```
