# NFT-007: 낯선 운영자(Conduit)를 통해 자산을 옮기는 마켓플레이스 주문에 서명할 시 경고

### Policy Definition (정책 정의)

> Seaport 주문이 알려진 마켓플레이스의 운영자가 아닌 낯선 운영자를 통해 토큰을 옮기는 경우 경고를 표시합니다.

Seaport에서 거래를 만들 때, 실제로 토큰을 옮기는 주체는 conduit이라는 운영자입니다. 주문에 서명하면 그 conduit에게 내 토큰을 가져갈 권한을 함께 넘깁니다. OpenSea처럼 널리 쓰이는 마켓은 검증된 conduit을 쓰지만, 악의적인 사용자는 자기가 통제하는 conduit을 끼워 넣은 주문에 서명하게 만듭니다. 주문이 알려진 마켓의 conduit 아닌 낯선 conduit를 거치면, 서명 전에 출처를 한 번 더 확인하도록 경고합니다.

#### Scope (적용 범위)

Seaport 주문 서명에 적용됩니다. 주문이 conduit을 거치고, 그 conduit이 허용 목록에 없는 낯선 키일 때만 발동합니다. conduit 없이 Seaport로 바로 옮기는 주문은 대상이 아닙니다.&#x20;

#### Audience (대상 사용자)

OpenSea 등 NFT 마켓플레이스를 사용하는 사용자

#### Used Data (판정에 사용될 데이터)

* 거래 마켓의 마켓플레이스 프로토콜 (seaport)
* conduit 키가 허용 목록(OpenSea conduit 키)에 들어 있는지

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("non-opensea-conduit-warn")
@severity("warn")
@reason("이 주문이 알려진 마켓플레이스가 아닌 낯선 운영자(conduit)를 통해 토큰을 옮깁니다 — 서명 전에 출처를 확인하세요")
forbid (
    principal,
    action == Marketplace::Action::"SignOrder",
    resource
)
when
{
    context.venue.name == "seaport" &&
    context.usesConduit == true &&
    !(["0x0000007b02230091a7ed01230072f7006a004d60a8d4e71d599b8104250f0000"].contains(context.conduitKey))
};
```
