# NFT-008: 받을 대금이 최저가보다 크게 낮은 NFT 리스팅 시 경고

### Policy Definition (정책 정의)

> Seaport 리스팅으로 받게 될 **대금이 컬렉션 최저가(floor)의 절반에도 못 미치는 경우** 경고를 표시합니다.

NFT를 팔려고 마켓플레이스에 리스팅하면, Seaport 주문에 서명하는 순간 "이 NFT를 이 가격에 넘기겠다"는 약속이 만들어집니다. 누군가 그 주문을 체결하면 정해둔 가격에 그대로 거래가 끝납니다. 피싱 사이트는 가격을 사실상 0에 가깝게 적어 둔 주문에 서명하게 만듭니다. 이렇게 리스팅된 NFT는 봇이 곧바로 사 가, 비싼 NFT를 푼돈에 빼앗기게 됩니다. 받을 대금을 컬렉션 최저가와 비교해, 최저가의 절반(5,000bp)보다도 낮으면 서명 전에 경고합니다.

#### Scope (적용 범위)

Seaport 주문 서명에 적용됩니다. 받을 대금이 컬렉션 바닥가보다 5,000bp(50%) 넘게 낮은 리스팅일 때 발동합니다.&#x20;

#### Audience (대상 사용자)

NFT를 마켓플레이스에서 판매를 하는 사용자

#### Used Data (판정에 사용될 데이터)

체결하려는 주문의 마켓플레이스 프로토콜

내놓는 NFT(`offer`) 및 해당 NFT컬렉션의 최저가

리스팅 한 금액

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("listing-proceeds-below-floor-warn")
@severity("warn")
@reason("받게 될 대금이 이 컬렉션 바닥가(floor)보다 훨씬 낮습니다 — 거의 공짜로 리스팅하는 게 아닌지 확인하세요")
forbid (
    principal,
    action == Marketplace::Action::"SignOrder",
    resource
)
when
{
    context.venue.name == "seaport" &&
    context has custom &&
    context.custom has proceedsBelowFloorBps &&
    context.custom.proceedsBelowFloorBps.greaterThan(decimal("5000.0000"))
};
```
