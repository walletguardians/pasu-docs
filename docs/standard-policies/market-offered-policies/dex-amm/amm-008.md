# AMM-008: 보유한 입력 토큰의 절반을 초과해 스왑할 시 경고

### Policy Definition (정책 정의)

> 한 번의 스왑에서 파는 토큰이 그 토큰 보유량의 절반을 초과하는 경우 경고합니다.

스왑은 들고 있던 토큰을 팔아 다른 토큰으로 바꾸는 거래입니다. 그런데 가진 입력 토큰의 절반이 넘는 양을 한 번에 파는 건, 비중을 줄이려는 의도된 거래일 수도 있지만 수량을 잘못 입력했거나 피싱 사이트가 지갑을 비우는 중일 때도 똑같아 보입니다. 파는 양이 보유량의 절반을 넘으면, 정말 이만큼 바꿀 것인지 서명 전에 확인하도록 경고합니다.

#### Scope (적용 범위)

DEX 스왑. 파는 입력 토큰의 수량이 해당 토큰 보유량의 50%(5,000bp)를 초과하는 경우에 적용됩니다. (Swap)

#### Audience (대상 사용자)

DEX에서 큰 금액을 한 번에 바꾸는 일이 잦은 사용자

#### Used Data (판정에 사용될 데이터)

파는 입력 토큰(`tokenIn`)의 수량(`direction.amountIn`)이 해당 토큰 보유량에서 차지하는 비중 (`holdingsBp` — 5,000bp = 50%, 서버 조회 결과)

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-fraction-of-holdings-warn")
@severity("warn")
@reason("이 스왑이 보유한 입력 토큰의 절반 넘게 팝니다 — 진행 전에 확인하세요")
forbid(principal, action == Amm::Action::"Swap", resource)
when {
  context has custom
  && context.custom has holdingsBp
  && context.custom.holdingsBp > 5000
};
```
