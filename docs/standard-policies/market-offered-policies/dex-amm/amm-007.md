# AMM-007: 스왑 금액이 $25,000을 초과할 시 경고

### Policy Definition (정책 정의)

> 한 번에 교환하는 토큰 금액이 미화 약 $25,000을 넘는 경우, 규모를 다시 확인하도록 경고합니다.

스왑은 가진 토큰을 다른 토큰으로 바꾸는 거래로, 금액이 클수록 결과를 되돌리기 어렵습니다. 금액을 한 자리 잘못 입력했거나, 거래량이 얕은 풀로 큰 물량을 밀어 넣어 가격이 크게 밀리거나(슬리피지), MEV 봇에 앞뒤로 끼이면 받는 양이 기대보다 줄어들 수 있습니다. 넣는 토큰을 시세로 환산한 값이 기준을 넘으면, 정말 이만한 규모로 교환할 것인지 서명 전에 한 번 더 따져보도록 경고합니다.

#### Scope (적용 범위)

DEX 스왑. 넣는 토큰(`tokenIn`)과 수량을 시세로 환산했을 때 $25,000을 넘는 경우에 적용됩니다. (Swap)

#### Audience (대상 사용자)

DEX에서 큰 금액을 한 번에 교환하는 사용자

#### Used Data (판정에 사용될 데이터)

넣는 토큰과 수량의 USD 환산 가치 (`swapUsd` — 시세 조회 결과)

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-usd-cap-warn")
@severity("warn")
@reason("고액 스왑입니다(명목가 약 $25,000 초과) — 규모를 확인하세요")
forbid(principal, action == Amm::Action::"Swap", resource)
when {
  context has custom
  && context.custom has swapUsd
  && context.custom.swapUsd.greaterThan(decimal("25000.0000"))
};
```
