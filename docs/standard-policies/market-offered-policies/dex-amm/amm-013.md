# AMM-013: 악성으로 분류된 라우터를 통해 스왑할 시 차단

### Policy Definition (정책 정의)

> 스왑이 가짜·악성으로 분류된 라우터나 어그리게이터를 거치는 경우, 거래가 나가지 않도록 **차단**합니다.

스왑은 보통 Uniswap이나 1inch 같은 라우터·어그리게이터 컨트랙트를 거쳐 체결됩니다. 그런데 정상 라우터를 흉내 낸 가짜 컨트랙트로 스왑을 유도하면, 받기로 한 토큰 대신 가치 없는 토큰을 떠넘기거나 입력 자금을 그대로 빼돌릴 수 있습니다. 스왑이 거치는 라우터 주소를 조회한 결과가 악성으로 분류되면, 거래를 막습니다.

#### Scope (적용 범위)

DEX 스왑(Swap). 거래가 거치는 라우터·어그리게이터가 악성으로 분류된 경우에 적용됩니다.

#### Audience (대상 사용자)

처음 보는 사이트에서 스왑하거나, 낯선 어그리게이터로 라우팅되는 거래를 자주 하는 사용자

#### Used Data (판정에 사용될 데이터)

스왑이 거치는 라우터 주소

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-venue-reputation-deny")
@severity("deny")
@reason("이 스왑이 가짜·악성 어그리게이터로 분류된 라우터를 경유해 차단했습니다")
forbid(principal, action == Amm::Action::"Swap", resource)
when {
  context has custom
  && context.custom has reputationFlagged
  && context.custom.reputationFlagged == true
};
```
{% endcode %}
