# AMM-009: 스왑이 풀 가격을 5% 넘게 움직일 시 경고

### Policy Definition (정책 정의)

> 이번 스왑이 **풀 가격을 5% 넘게 움직이는** 경우, 너무 비싸게 사고파는 건 아닌지 다시 확인하도록 경고합니다.

스왑은 한 번에 많은 양을 바꾸면 그만큼 풀 안의 가격이 밀립니다. 이 밀림이 클수록 시세보다 불리하게 체결되는데, 보통은 유동성이 얕은 마켓이거나 한 번에 너무 큰 금액을 바꾸려 할 때 나타납니다. 스왑이 풀 가격을 5% 넘게 움직이면, 정말 이 가격에 체결할 것인지 서명 전에 확인하도록 경고합니다.

#### Scope (적용 범위)

DEX 스왑(Swap). 이번 거래의 가격 영향이 5%를 초과하는 경우에 적용됩니다.

#### Audience (대상 사용자)

거래량이 적은 토큰이나 신생 풀에서 스왑하는 사용자

#### Used Data (판정에 사용될 데이터)

* 스왑이 풀 가격을 움직이는 폭

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-price-impact-cap-warn")
@severity("warn")
@reason("이 스왑이 풀 가격을 5% 넘게 움직입니다 — 호가가 얇은 마켓에서 비싸게 사는 것일 수 있습니다")
forbid(principal, action == Amm::Action::"Swap", resource)
when { context.priceImpactBp > 500 };
```
