# AMM-012: 스왑으로 받을 토큰이 스캠일 경우  경고

### Policy Definition (정책 정의)

> 스왑으로 **받게 될 토큰**이 알려진 스캠 토큰으로 분류된 경우, 진행하기 전에 다시 확인하도록 경고합니다.

받는 쪽 토큰이 멀쩡해 보여도 실제로는 팔 수 없게 막아두거나, 보유량을 빼가는 스캠 컨트랙트인 경우가 있습니다. 이런 토큰은 한 번 받고 나면 다시 팔거나 빼내기 어려워, 스왑에 넣은 자산을 그대로 잃을 수 있습니다. 받게 될 토큰이 스캠으로 분류돼 있으면, 정말 이 토큰으로 바꿀 것인지 서명 전에 한 번 더 확인하도록 경고합니다.

#### Scope (적용 범위)

DEX 스왑(Swap). 스왑으로 받게 될 토큰이 스캠으로 분류된 경우에 적용됩니다.

#### Audience (대상 사용자)

DEX를 사용하는 모든 사용자

#### Used Data (판정에 사용될 데이터)

* 받게 될 토큰의 스캠 분류 여부

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-output-token-reputation-warn")
@severity("warn")
@reason("사려는 토큰이 알려진 스캠 토큰으로 분류됐습니다 — 진행 전에 확인하세요")
forbid(principal, action == Amm::Action::"Swap", resource)
when {
  context has custom
  && context.custom has reputationFlagged
  && context.custom.reputationFlagged == true
};
```
{% endcode %}
