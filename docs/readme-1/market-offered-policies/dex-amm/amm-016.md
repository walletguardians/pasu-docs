# AMM-016: 전체 유동성이 적은 토큰을 매수할 시 경고

### Policy Definition (정책 정의)

> 스왑하려는 **토큰의 거래 가능 유동성이 거의 없는** 경우, 진행하기 전에 다시 확인하도록 경고합니다.

토큰을 살 수는 있어도, 그 토큰을 받아줄 유동성이 없으면 되팔 때 제값을 받지 못하거나 아예 팔지 못할 수 있습니다. 사려는 토큰의 전체 유동성이 기준에 못 미치면 서명 전에 확인하도록 경고합니다.

#### Scope (적용 범위)

받게 될 토큰의 전체 유동성이 기준에 못 미치는 경우에 적용됩니다.

#### Audience (대상 사용자)

유동성이 얕은 신규, 소형 토큰을 스왑으로 매수하는 사용자

#### Used Data (판정에 사용될 데이터)

받게 될 토큰의 전체 풀 유동성(`buyTokenLiquidityUsd`)이 기준보다 낮은지 확인합니다.

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("swap-buy-thin-token-warn")
@severity("warn")
@reason("스왑하려는 토큰의 전체 유동성이 매우 적습니다 — 대상 토큰을 확인하세요")
forbid(principal, action == Amm::Action::"Swap", resource)
when {
  context has custom
  && context.custom has buyTokenLiquidityUsd
  && context.custom.buyTokenLiquidityUsd.lessThan(decimal("50000.0000"))
};
```
{% endcode %}
