# AMM-018: 인텐트 주문의 유효 기간이 지나치게 길 시 경고

### Policy Definition (정책 정의)

> 인텐트 기반 거래(UniswapX·CowSwap 등)의 주문의 유효 기간이 기준보다 긴 경우, 서명 전에 다시 확인하도록 경고합니다.

서명한 주문은 정해진 유효 기간 동안 언제든 solver가 체결할 수 있습니다. 이 기간이 너무 길면, 시세가 나에게 불리하게 바뀐 뒤에 뒤늦게 체결돼 손해를 볼 수 있습니다. 주문의 유효 기간이 기준을 넘으면, 그렇게 오래 열어둘 필요가 있는지 서명 전에 확인하도록 경고합니다.

#### Scope (적용 범위)

주문의 유효 기간이 기준을 초과하는 경우에 적용됩니다.

#### Audience (대상 사용자)

인텐트 기반 거래를 사용하는 사용자

#### Used Data (판정에 사용될 데이터)

주문의 유효 만료 시각(`validUntil`)이 기준을 넘는지 확인합니다.

#### Policy in Code

{% code title="policy.cedar" %}
```solidity
@id("intent-order-far-deadline-warn")
@severity("warn")
@reason("주문의 유효기간이 기준을 넘습니다 - 유효기간을 확인하세요")
forbid(principal, action == Amm::Action::"SignIntentOrder", resource)
when {
  context has custom
  && context.custom has nowTs
  && context.validUntil > context.custom.nowTs + 604800
};
```
{% endcode %}
