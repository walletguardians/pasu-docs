# 마켓플레이스에서 정책 찾기·설치하기

마켓플레이스는 미리 만들어진 정책과 패키지를 찾아 설치하는 정책 허브입니다. 직접 Cedar를 쓰지 않고도 검증된 보호 규칙을 바로 적용할 수 있어요.

## 정책 vs 패키지(세트)

| 종류 | 설명 |
|------|------|
| **정책 (policy)** | 단일 규칙 (예: "무제한 승인 차단") |
| **패키지/세트 (set)** | 여러 정책을 묶은 모음 (예: "DEX 안전 팩") |

## 둘러보기

### 랜딩 화면 (`/market`)

* **공식 패키지** — 설치수 기준 상위 패키지 (verified 배지 ✓)
* **추천 영역** — 최근 7일 인기 카테고리 / 보호 공백(coverage gap) 추천
* **카테고리 그리드** — 도메인별 타일(자산·프로토콜 유형)과 정책/패키지 수
* **인기 정책 사이드바** — 최근 설치 기준 Top 10

### 목록 화면 (`/market?view=list`)

* **검색** — 이름·slug·한 줄 설명 매칭, 최근/인기 검색어 제공
* **카테고리 다중 선택** (합집합 필터)
* **정렬** — 인기순(popular) / 최신순(new) / 평점순(rating)
* **페이지네이션** — 페이지당 24개, 패키지 → 정책 순으로 구획

## 상세 페이지 (`/market/:slug`)

정책 상세에서 확인할 수 있는 것:

* **게시자 등급(tier)** — Official(공식, ✓) / Verified / Community
* 표시 이름·slug·버전·설치수·평점(★ 평균, 리뷰 수)
* **설치 버튼** — `받기` 또는 이미 설치했다면 `설치됨`
* **문서** — 정의 / 범위 / 대상 / 데이터 (정책이 무엇을·어디까지·누구를 위해·어떤 데이터로 동작하는지)
* **정책 카드** — 강도(deny=빨강, warn=주황), 카테고리 태그, 한 줄 설명
* **리뷰** — 평점(1~5)과 후기
* 패키지인 경우: 포함된 정책 수 + 대표 카테고리 태그

## 설치하기

1. 상세 페이지에서 **받기(Install)** 클릭
2. **파라미터 입력** — 정책에 빈칸(hole)이 있으면 채웁니다 (예: 경고 기준 금액)
3. **적용 범위(scope) 선택** — 어떤 지갑/패키지에 넣을지 지정
4. 설치하면 정책이 에디터로 복사되고 확장프로그램에 로컬 저장됩니다(`ps2:*`)

> 설치는 서버에 이벤트로 기록되지만(설치수 집계용), 정책 자체와 평가는 모두 기기 내부에서 일어납니다.

## 평가·신고

* **리뷰 작성** — 설치한 버전에 평점과 후기를 남길 수 있습니다
* **도움됨 투표** — 유용한 리뷰에 투표
* **신고(Report)** — 위험·오해 소지·스팸 등으로 정책/리뷰 신고 (모더레이션 큐로 전달)
* **워치(Watch)** — 관심 정책을 찜해두기
