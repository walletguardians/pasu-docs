# 정책 허브에서 정책 찾기·설치하기

정책 허브는 미리 만들어진 정책과 패키지를 찾아 설치하는 곳입니다. 직접 정책(Cedar 코드)를 쓰지 않고도 검증된 보호 규칙을 바로 적용할 수 있어요.

## 정책 vs 패키지(세트)

| 종류               | 설명                           |
| ---------------- | ---------------------------- |
| **정책 (policy)**  | 단일 규칙 (예: "무제한 승인 차단")       |
| **패키지(package)** | 여러 정책을 묶은 모음 (예: "DEX 안전 팩") |

## 둘러보기

### Market Home

* **공식 패키지** — 설치수 기준 상위 패키지 (verified 배지 ✓)
* **추천 영역** — 최근 7일 인기 카테고리 / 보호 공백(coverage gap) 추천
* **카테고리 그리드** — 도메인별 타일(자산·프로토콜 유형)과 정책/패키지 수
* **인기 정책 사이드바** — 최근 설치 기준 Top 10

### 상세 페이지

정책 상세에서 확인할 수 있는 것:

* **게시자 등급(tier)** — Official(공식, ✓) / Verified / Community
* **설치 버튼** — `받기` 또는 이미 설치했다면 `설치됨`
* **문서** — 정의 / 범위 / 대상 / 데이터 (정책이 무엇을·어디까지·누구를 위해·어떤 데이터로 동작하는지)
* **정책 카드** — 강도, 카테고리 태그, 한 줄 설명
* **리뷰** — 평점과 후기

## 설치하기 <지갑 전용으로 받기>

1. 상세 페이지에서 **받기(Install)** 클릭
2. 지갑 전용 설정으로 받기 선택
3. 정책이 설치될 지갑 및 패키지 선택
4. **파라미터 입력** — 정책에 빈칸(hole)이 있으면 값 입력 가능
5. 정책이 에디터 및 해당 지갑의 패키지에 저장되고 사용 가능

## 설치하기 <라이브러리로 받기>

1. 상세 페이지에서 **받기(Install)** 클릭
2. 라이브러리로 받기 선택
3. 정책 템플릿이 설치될 폴더 선택
4. **파라미터 입력** — 정책에 빈칸(hole)이 있으면 값 입력 가능
5. 정책 템플릿이 에디터의 라이브러리에 저장되고 해당 템플릿을 자유롭게 수정해서 패키지에 적용 가능

## 평가·신고

* **리뷰 작성** — 설치한 버전에 평점과 후기를 남길 수 있습니다
* **도움됨 투표** — 유용한 리뷰에 투표
* **신고(Report)** — 위험·오해 소지·스팸 등으로 정책/리뷰 신고 (모더레이션 큐로 전달)
* **워치(Watch)** — 관심 정책을 찜해두기
