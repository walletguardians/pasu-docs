# 개인정보 처리방침

> 운영주체: **Wallet Guardians** · 최종 수정일: **2026-06-28**
> 정식 공개(웹스토어 등록) 전 법무 검토를 권장합니다. 기술적 상세는 [데이터 흐름](data-flows.md)·[보안 모델](security-model.md)을 참고하세요.

DAMBI는 **"판정은 기기 안에서, 데이터는 최소한으로"** 를 원칙으로 합니다. 트랜잭션/서명 평가는 확장프로그램 내부 WASM에서 끝나며, 서버 전송이 필요하지 않습니다.

## 다루는 데이터

**기기에만 저장 (서버 전송 없음)**: 설치한 정책, 판정 히스토리, 설정. (확장 로컬 저장소 `ps2:*` / `chrome.storage`)

**서버와 오가는 데이터** (로그인·동기화·정책 허브 사용 시)

| 데이터 | 목적 |
|--------|------|
| 계정 이메일 | 인증·세션(JWT) |
| 지갑 주소·상태 | 잔고/상태 컨텍스트, 예측 |
| remote facts 요청 파라미터 | 외부 사실 조회(예: USD 가치) |
| 정책 허브 활동(게시·리뷰·신고) | 허브 운영 |

**LLM 보조 초안 (선택)**: 정책 초안 생성 시 입력 내용이 **OpenAI**로 전송됩니다. API 키는 브라우저에만 저장되며 우리 서버를 거치지 않습니다. ([OpenAI 정책](https://openai.com/policies/))

## 사용 목적

위험 평가·경고/차단(기기 내), 인증·세션 유지, 지갑 상태 기반 예측, 정책 허브 운영. **광고·판매 목적으로 사용하지 않습니다** (Chrome 웹스토어 Limited Use 준수).

## 제3자 / 외부 서비스

| 서비스 | 용도 |
|--------|------|
| Google OAuth | 로그인 |
| Google Cloud (GCP) | 서버 호스팅 |
| 레지스트리 (registryV2 / registry-api) | 디코딩용 어댑터·토큰 메타 |
| CoinGecko · Chainlink | 토큰 가격 |
| Etherscan | 지갑 토큰 탐색 (선택) |
| Chainalysis | 제재 주소 검사 (선택) |
| Alchemy | NFT 바닥가 (선택) |
| OpenAI | LLM 정책 초안 (선택, 브라우저에서 직접 호출) |

## 보유·삭제·권리

* **기기 로컬 데이터**: 확장프로그램 삭제 또는 초기화 시 제거됩니다.
* **서버 데이터**: 계정이 있는 동안 보관하며, **계정·지갑 삭제 또는 요청 시 제거**합니다.
* 열람·삭제 요청은 아래 문의 채널로 연락 주세요.

## 보안 · 문의

* 판정은 기기 내에서 처리되어 트랜잭션 내용의 외부 노출을 최소화합니다. 레지스트리 번들은 ECDSA(P-256) 서명으로 검증됩니다. ([보안 모델](security-model.md))
* 문의·취약점 제보: **GitHub** [walletguardians](https://github.com/walletguardians) 의 Issues 및 Security Advisories.
